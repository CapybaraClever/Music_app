# Музыкальный архив (Music_app)

Десктопное GUI-приложение для учёта музыкальных композиций, написанное на **C++17** с использованием фреймворка **Qt Widgets**.

---

## Структура проекта

```text
Music_app/
├── main.cpp                 # Точка входа в приложение
├── Music_app.h / .cpp       # Главное окно: меню, тулбар, таблица, слоты и логика
├── addsongdialog.h / .cpp   # Диалог добавления песни (QDialog) с валидацией ввода
├── song.h                   # Структура Song + операторы для QDataStream
└── Music_app.vcxproj        # Файл проекта для Visual Studio
````

---

## Требования для запуска

- **ОС:** Windows
- **Среда разработки:** Visual Studio
- **Фреймворк:** Qt 6.x (компилятор MSVC)
- **Плагин:** Установленное расширение _Qt Visual Studio Tools_

## Сборка и запуск

1. Откройте файл проекта `Music_app.vcxproj` (или файл решения `Music_app.sln`) в Visual Studio.
    
2. Дождитесь завершения индексации файлов проектом.
    
3. Нажмите комбинацию клавиш **Ctrl + F5** (или кнопку «Локальный отладчик Windows» на верхней панели).

## Функциональность приложения

| **Функция**          | **Описание**                                                                        |
| -------------------- | ----------------------------------------------------------------------------------- |
| **Новый**            | Очистить текущий список композиций (с предупреждением о несохраненных данных).      |
| **Открыть**          | Загрузить существующий архив из бинарного файла `.msa`.                             |
| **Сохранить**        | Перезаписать текущий открытый файл `.msa` (сериализация через QDataStream).         |
| **Сохранить как...** | Выбрать новое место и имя для сохранения файла архива.                              |
| **Добавить**         | Открытие диалогового окна со строгой валидацией полей «Название», «Автор», «Год».   |
| **Удалить**          | Удаление выбранной песни из таблицы с обязательным кастомным окном подтверждения.   |
| **Инфо**             | Вывод подробных данных о песне (также открывается по двойному клику по строке).     |
| **Найти старейшую**  | Автоматический поиск, выделение в таблице и показ инфо о песне с минимальным годом. |
| **Сортировка А→Я**   | Сортировка всего списка по названию композиции по возрастанию (locale-aware).       |
| **Сортировка Я→А**   | Сортировка всего списка по названию композиции по убыванию.                         |
| **О программе**      | Однокликовая кнопка меню с кастомным интерфейсом и... тсс, зайдёте и узнаете        |

## Формат сохранения файлов (`.msa`)

Программа использует защищенный бинарный формат данных, сериализуемый через встроенный поток `QDataStream` (версия совместимости `Qt_6_0`). Это защищает файлы базы данных от прямого несанкционированного редактирования через сторонние текстовые редакторы.

**Спецификация заголовка и тела файла:**

Plaintext

```
[quint32] Magic number  = 0x4D534100  ("MSA\0" для идентификации формата)
[quint32] File version  = 1
[qint32]  Количество записей = N
[Song×N]  Последовательный блок данных: Название (QString) + Автор (QString) + Год (int)
```

При вызове операции «Открыть» приложение выполняет пошаговую валидацию структуры файла. Если контрольная сигнатура (Magic number) или версия формата не совпадают с ожидаемыми, чтение данных прерывается с выводом ошибки `QMessageBox::warning`, что полностью исключает падение программы из-за попытки чтения битых или некорректных файлов.
